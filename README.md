# Selenium Login Automation Framework

This project demonstrates Selenium automation using Python with Pytest framework.

## Features
- Page Object Model (POM)
- Pytest fixtures
- Positive and Negative test cases
- Explicit waits
- Organized folder structure

## Technologies Used
- Python
- Selenium
- Pytest

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   pytest
